\begin{itemize}
	\item The origin is between LPA \& RPA
	\item The $\vect{x}$ axis goes through nasion.
	\item The $\vect{y}$ goes through LPA, orthogonal to $\vect{x}$
	\item The $\vect{z}$ axis is orthogonal to both $\vect{x}$ \& $\vect{y}$
\end{itemize}
